# 💻 Object-Oriented Programming in C++ — Complete Programs Collection

This repository contains a complete collection of programs focused on **Object-Oriented Programming (OOP) in C++**. It is designed to help learners understand core OOP concepts through practical examples and build strong programming skills using C++.


## 📚 About This Repository

This repository is created for learning and practicing **OOP concepts in C++**. It includes simple to advanced programs demonstrating how object-oriented principles are implemented in real code.

## 🚀 Topics Covered

### 🔹 Core OOP Concepts

* Classes and Objects
* Constructors and Destructors
* Encapsulation
* Abstraction
* Inheritance
* Polymorphism

### 🔹 Intermediate Concepts

* Function Overloading
* Operator Overloading
* Friend Functions
* Static Members
* This Pointer

### 🔹 Advanced Concepts

* Virtual Functions
* Runtime Polymorphism
* Abstract Classes
* Multiple Inheritance
* Hierarchical Inheritance

## 🧪 Sample Programs

* Class and object implementation
* Constructor & Destructor examples
* Inheritance-based programs
* Polymorphism demonstrations
* Function & operator overloading
* File handling programs
* Student management system (basic)

## ⚙️ How to Run

1. Install a C++ compiler (G++ recommended)

2. Compile any program:

```id="k9d2qs"
g++ program.cpp -o program
```

3. Run the executable:

```id="v1p7mz"
./program
```
